# tpmp-lab2-task4

## Overview
Программа для обработки текстовых файлов с данными о военнослужащих.
Создаёт файл с данными, выводит содержимое, фильтрует военнослужащих
в звании "лейтенант" и сохраняет результат в новый файл.
